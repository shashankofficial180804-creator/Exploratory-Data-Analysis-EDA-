# Exploratory-Data-Analysis-EDA-
Exploratory Data Analysis (EDA) project using Python on Netflix Movies &amp; TV Shows and Iris datasets. Includes data cleaning, visualization, outlier detection, correlation analysis, and feature insights using Pandas, Matplotlib, and Seaborn as part of an AI &amp; ML internship task.

### Objective
To perform Exploratory Data Analysis on:
- Netflix Movies and TV Shows Dataset
- Iris Dataset

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

### Key Tasks
- Distribution analysis using histograms
- Categorical analysis using count plots
- Outlier detection using box plots
- Correlation analysis using heatmaps
- Feature importance identification

### Files Included
- EDA_Task_3.ipynb
- Visual_Insights_Report.md
- netflix_titles.csv
- iris.csv

### Outcome
Gained strong understanding of data patterns, feature behavior, and visualization techniques.
